Installation
==============

Requirements:

* ``python >=3.9``
* ``pandas >= 1.5.0``
* ``click >= 8.1.7``

TCRconvert runs on Windows, macOS, and Linux.

Install from GitHub using ``pip``:

.. code-block:: console

   $ pip install git+https://github.com/seshadrilab/tcrconvert


Or clone this repo and from the top-level folder run:

.. code-block:: console

   $ pip install .
